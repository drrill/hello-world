## House rules for the rewrites

- No em dashes. Commas, semicolons, parentheses, or periods instead.
- No "it's not X, it's Y" parallelism.
- No "not just X but Y" constructions.
- Short paragraphs, specific references, one key point per post.
- Light, unobtrusive supporting links to related thinking.
